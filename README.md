# How to Use (For Normal Users)

This script is designed for **regular Facebook users** who want to cancel follow requests or unfollow accounts in bulk.
You **do not need to be a developer** to use it.

No software installation, browser extensions, or coding knowledge is required.

## Step-by-Step Instructions

### Step 1: Open Facebook

1. Open **Google Chrome**
2. Log in to your Facebook account
3. Go to your **Following** or **Followers** list
   (Where you see the *Cancel request* or *Unfollow* buttons)

---

### Step 2: Open the Browser Console

1. Press **F12** on your keyboard
   *(or right-click → Inspect)*
2. Click the **Console** tab at the top

You will see a blank area where you can paste code.

---

### Step 3: Copy the Script

1. Open this link in a new tab:

[facebook-auto-unfollow-cancel-request-script](https://github.com/sksshouvo/facebook-auto-unfollow-cancel-request-script)

2. Open the script file
3. Copy **all** of the script code

---

### Step 4: Paste and Run

1. Paste the copied script into the **Console**
2. Press **Enter**

The script will now:

* Scroll to the bottom of the list
* Find all **Cancel request / Unfollow** buttons
* Click them **one by one**
* Wait **5 seconds** between each action

---

## While the Script Is Running

* Keep the Facebook tab **open and visible**
* Do **not** switch accounts
* Do **not** close the tab
* Do **not** run multiple scripts at the same time

The process may take time depending on how many requests you have.

---

## How to Stop the Script

You can stop the script **at any time**.

### Option 1: Stop Without Refreshing (Recommended)

Paste this into the Console and press **Enter**:

```
window.__STOP_UNFOLLOW__ = true;
```

### Option 2: Refresh the Page

Simply refresh the Facebook page to stop everything immediately.
