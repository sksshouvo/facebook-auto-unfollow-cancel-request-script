# facebook-auto-unfollow-cancel-request-script
This script will help you to unfollow or cancel requests people from your facebook automatically.

# Follow below instruction to run the script
1. Log in to your facebook account.
2. Got to your following list page
3. Press Clt + i to open inspect tab
4. Then go to console tab
5. Then run 'allow pasting'
6. Then copy the file's code index.js
7. Then paste into that console and hit Enter button to run the script and withness the magic.
